== Description ==

== Installation ==

1. Upload the plugin iPermisson files to the `/wp-content/plugins/` directory
2. Activate the iPermisson plugin through the 'Plugins' menu in WordPress.
3. Select iPermisson from your Wordpress Admin Menu
